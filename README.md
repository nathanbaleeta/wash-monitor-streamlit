# wash-monitor-streamlit

### Pre-requisites
- Python 3.xx+
- [Streamlit](https://streamlit.io/)
- Pandas


### Quick Setup
The project uses [Pip](https://pypi.org/project/pip/) to keep track of its dependencies. To setup follow instructions below:

```bash
git clone git@github.com:nathanbaleeta/wash-monitor-streamlit.git

cd wash-monitor-streamlit

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt
```

### Launch app
To bootstrap the app 
```
streamlit run app.py
```
